# Pseudo-classes

Keywords added to selectors to style selected elements in a specific state. `:hover` is probably
one of the most common examples.

## Syntax

```css
selector:pseudo-class {
    property: value;
}
```

## Example

```css
a:hover {
    color: red;
}
```
